# 神话人格 · 20位角色数据文档

> 版本 v1.1 · 2026-06-10  
> 每位角色包含：身份 / 特质标签 / 性格解读 / 神话故事摘要 / 经典名言 / 6维人格数值

---

## 数据字段说明

| 字段 | 说明 |
|------|------|
| **身份** | 该神祇在神话体系中的核心定位，15字以内 |
| **特质** | 5个性格标签，2-4字，代表其最鲜明的人格特征 |
| **性格解读** | 第二人称（"你"），让用户产生共鸣，120-160字 |
| **神话故事** | 最具代表性的故事，强调画面感与人格联系，120-160字 |
| **名言** | 代表其世界观的一句话，20-40字，用引号包裹 |
| **6维数值** | 权威 / 智慧 / 激情 / 自由 / 深沉 / 守护，0-100 |
| **命运羁绊** | 与其他神祇的关系描述，每条15字以内；由算法自动取相似度最高2位（相契）和最低2位（张力），此处只需提供文案；每位神祇需为全部19位对应神祇各写一条关系描述 |

## 01 · 宙斯 Zeus ⚡

**身份**：奥林匹斯之王 · 天空与雷电之神

**特质**：领袖 · 权威 · 果断 · 骄傲 · 慷慨

**性格解读**  
你身上有一种与生俱来的领导气场——人们自然而然地听从你，不是因为你强迫，而是因为你确实能撑起局面。你渴望秩序，厌恶混乱，对自己的判断充满自信，有时甚至过于自信。你善于在最混乱的局面里做出快速决策，并且往往是对的。你有慷慨的一面，愿意庇护弱者，但你的自尊心极强，一旦权威受到挑战，怒火会来得毫无预兆。你的弱点不是能力，而是执念——对控制的执念，对自己永远正确的执念。

**神话故事**  
泰坦之战是宙斯最重要的时刻。他的父亲克洛诺斯吞噬了所有兄姐，只有他被母亲瑞亚藏于克里特岛秘密抚养。成年后，宙斯孤身潜入父亲宫殿，用智慧而非蛮力使克洛诺斯吐出众兄姐，随后领导十年漫长战争，最终将泰坦神族囚于塔耳塔罗斯。这场胜利不仅是力量的胜利，更是意志的胜利——他从一个被遗弃的婴儿，成为了整个宇宙的主宰。

**名言**  
「雷鸣不只是愤怒，更是宣告：这片天地，有人在守护。」
